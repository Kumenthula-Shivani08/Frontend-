4)write a python program to find the duplicate items in list ?
code:
# Sample list
my_list = [1, 2, 3, 4, 3, 5, 6, 1, 7, 8, 5]

# Find duplicates using a simple loop
duplicates = []
seen_items = set()

for item in my_list:
    if item in seen_items:
        duplicates.append(item)
    else:
        seen_items.add(item)

print("Duplicates found:", duplicates)
